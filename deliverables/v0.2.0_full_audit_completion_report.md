## 二、P0-1 standings 错位 · 根因 + 修复

### 根因（深度）

`wc26.ir /get/groups` 的 `team_id` 是**线性编号 1-48**（按 A-L 组顺序：1-4 A 组，5-8 B 组，9-12 C 组……），与我们的 `Team.id`（SQLite 自增 + FIFA 抽签顺序）**不一致**。

| 队 | Team.id | 真实 group | wc26 写入的 group（错）| 错位原因 |
|---|---|---|---|---|
| MEX | 1 | A | A ✓ | 1-4 是 A 组（巧合对齐）|
| USA | 7 | D | B | wc26 把 7-10 当 B 组 |
| QAT | 9 | B | C | wc26 把 9-12 当 C 组 |
| ESP | 25 | H | G | wc26 把 25-28 当 G 组 |
| ENG | 43 | L | K | wc26 把 43-46 当 K 组 |

**18 队**（id 7, 8, 9, 10, 13, 14, 19, 20, 21, 22, 25, 26, 31, 32, 43, 44, 47, 48）每队被写错 1 次，导致 DB 里 66 条 standing（48 真 + 18 错），**12 重复**。

### 修复

**1. 代码修复**（`app/services/worldcup26_sync.py`）

`sync_standings` 复用 `sync_matches` 已有的 `wc26_id → fifa_code` 映射模式：

```python
# 拉 wc26 teams 构建 wc26_id → fifa_code 映射
wc26_id_to_fifa: dict[int, str] = {}
wc26_teams_data = fetch_json("/get/teams")
if wc26_teams_data:
    for t in wc26_teams_data.get("teams", []):
        tid = _to_int_or_none(t.get("id"))
        fc = (t.get("fifa_code") or "").strip()
        if tid and fc:
            wc26_id_to_fifa[tid] = fc

# 预加载我们 Team 表：fifa_code → Team.id
team_by_fifa: dict[str, int] = {t.fifa_code.upper(): t.id for t in db.query(Team).all() if t.fifa_code}

# 关键：wc26_id → fifa_code → 我们的 Team.id
fifa = wc26_id_to_fifa.get(wc26_id)
team_id = team_by_fifa.get(fifa.upper())
```

**2. DB 清理**


### 附带 bug 修复

`prediction_cache.py` 的 `age = (datetime.now(timezone.utc) - cache.generated_at).total_seconds()` 会抛 `TypeError: can't subtract offset-naive and offset-aware datetimes`，因为 DB 存的 `DateTime` 是 naive（SQLite 无 timezone）。修法：

```python
generated_at = cache.generated_at
if generated_at.tzinfo is None:
    generated_at = generated_at.replace(tzinfo=timezone.utc)
age = (datetime.now(timezone.utc) - generated_at).total_seconds()
```

## 四、P1-1 API team_id 改 path · 修复

### 问题

`/api/teams/{team_id}` 仅支持 int ID。客户端按 FIFA 3 字母代码（如 `BRA`）调会 422（int 解析失败）。

### 修复（`app/routers/teams.py`）

```python
def _resolve_team(db: Session, code: str) -> Team:
    if code.isdigit():
        team = db.query(Team).filter(Team.id == int(code)).first()
    else:
        team = db.query(Team).filter(Team.fifa_code == code.upper()).first()
    if not team:
        raise HTTPException(status_code=404, detail=f"球队不存在: {code}")
    return team

@router.get("/teams/{team_code}", response_model=TeamOut)
def get_team(team_code: str, db: Session = Depends(get_db)) -> Team:
    """球队详情（支持 team_id 数字或 FIFA 代码如 'BRA'）."""
    return _resolve_team(db, team_code)

@router.get("/teams/{team_code}/matches", response_model=List[MatchOut])
def team_matches(team_code: str, db: Session = Depends(get_db)) -> List[Match]:
    team = _resolve_team(db, team_code)
    return db.query(Match).filter((Match.home_team_id == team.id) | (Match.away_team_id == team.id)).all()
```
