### 4.6 h2h.py · 9 队非参赛队 fallback（+18 行）

**Bug**：`/api/h2h/BRA/SRB` 返 404（h2h 路由要求 `db.query(Team).filter_by(fifa_code==code)` 命中，但 SRB 不在 48 队 teams 表里）。但 `h2h_historical_matches` 表里有 SRB 2 场，h2h-opponents API 也列了 SRB。

**修法**：

```python
team1 = db.query(Team).filter(Team.fifa_code == code1_u).first()
team2 = db.query(Team).filter(Team.fifa_code == code2_u).first()
if not team1 or not team2:
    def _fallback(code: str) -> Dict:
        return {
            "fifa_code": code,
            "name_zh": code,  # 没中文名就显示代码
            "name_en": code,
            "flag_emoji": "🏳️",  # 灰色 fallback
            "group_name": None,
        }
    if not team1 and not team2:
        raise HTTPException(status_code=404, detail=f"球队 {code1_u} 与 {code2_u} 都不存在")
    if not team1:
        team1 = type("T", (), _fallback(code1_u))()
    if not team2:
        team2 = type("T", (), _fallback(code2_u))()
```

**修复后**：`/api/h2h/BRA/SRB` 返 200（summary 2 场 0 平 0 客胜），详情页 H1 显示 🇧🇷 巴西 VS 🏳️ SRB（SRB 队用 fallback 显示）。
