## 🆕 新增 API 行为

```
GET /api/simulator/tournament
  ?simulations=10000      # [100, 50000], 默认 10000
  ?model=blend            # blend | elo | glicko2, 默认 blend
  ?return_top_n=8         # top N 对阵频率
  ?seed=42                # 随机种子
  ?refresh=0              # 新增:1=强制重算,0=优先缓存
```

**缓存键**: `(model, n_sims, seed)`
**TTL**: 6 小时
**覆盖写**: 同键只保留最新一条

## 🧠 缓存命中逻辑

if not refresh:
    cached = load_mc_cache(db, model, n_sims, seed)
    if cached:
