## 背景

你表示不希望共享服务器 SSH 权限。以下是几种不需要把服务器密码/密钥交给我的部署渠道，按推荐程度排序。

---

## 渠道一：用户自助脚本部署（推荐，最简单）

**思路**：我提供完整的部署脚本和配置文件，你自己买服务器、上传代码、运行脚本。

**已准备**：
- `scripts/deploy_tencentcloud.sh`：一键安装 Python、Nginx、SSL，配置 Systemd
- `scripts/wc2026.service`：Systemd 服务模板（内嵌在脚本中）
- `scripts/nginx_wc2026.conf`：Nginx 配置模板（内嵌在脚本中）

**你需要做的事**：
1. 购买腾讯云 Lighthouse（2C2G 起步，约 50-70 元/月）
2. 域名解析到服务器 IP
3. 把项目代码上传到服务器 `/opt/wc2026`
4. 修改脚本里的 `DOMAIN` 和 `ADMIN_TOKEN`
5. 执行：`sudo bash scripts/deploy_tencentcloud.sh`

**优点**：
- 不需要给我 SSH
- 成本低，完全可控
- 脚本可复用，以后升级也简单

**缺点**：
- 需要你动手执行几条命令
- 首次部署约 10-15 分钟

---

## 渠道二：Docker Compose 部署（最便携）

**思路**：把应用打包成 Docker 镜像，通过 docker-compose 一键启动。

**已准备**：
- `scripts/Dockerfile`
- `scripts/docker-compose.yml`
- `scripts/deploy_docker.sh`

**你需要做的事**：
1. 购买任意支持 Docker 的服务器（腾讯云 Lighthouse、阿里云 ECS、Vultr、DigitalOcean 等）
2. 安装 Docker + Docker Compose
3. 上传项目代码
4. 执行：`sudo bash scripts/deploy_docker.sh`

**优点**：
- 环境隔离，换服务器只需复制代码和 volume
- 未来升级方便：`docker compose down && docker compose up -d`
- 不依赖特定云平台

**缺点**：
- 需要了解一点 Docker
- 仍需一台服务器

## 渠道三：Cloudflare Tunnel（零服务器，免费）

**思路**：在你本地电脑运行 cloudflared，把本地 `127.0.0.1:8000` 暴露到公网 URL。

**你需要做的事**：
1. 注册 Cloudflare 账号
2. 安装 cloudflared：`https://developers.cloudflare.com/cloudflare-one/connections/connect-networks/downloads/`
3. 运行：`cloudflared tunnel --url http://127.0.0.1:8000`
4. 获得一个 `https://xxx.trycloudflare.com` 临时域名

**优点**：
- 完全免费
- 不需要买服务器
- 5 分钟搞定

**缺点**：
- 你电脑必须一直开机
- 域名临时，每次重启会变
- 不满足“持续运行”要求（除非电脑 24 小时开机）
- 仅供临时演示

## 渠道四：海外 PaaS（Render / Railway / Fly.io）

**思路**：把代码推送到 GitHub，连接 PaaS 平台自动部署。

**适用性评估**：
- ❌ SQLite：平台文件系统通常只读或 ephemeral，数据会丢失
- ❌ APScheduler：平台会休眠/重启，后台任务不稳定
- ✅ 需要改造：改用 PostgreSQL + 外部定时任务（如 cron job / scheduler service）

**结论**：当前架构不适合直接部署到 PaaS，需要额外改造，不推荐作为首选。

---

## 渠道五：腾讯云 SCF + COS + Timer（Serverless 改造）

**思路**：
- API 用腾讯云云函数 SCF
- 数据用云数据库 + COS 对象存储
- 同步任务用 SCF 定时触发器

**结论**：
- 改造成本高（需要拆分同步任务、替换 SQLite、适配 Serverless 运行模式）
- 费用按调用量计费，适合高并发但不适合长期低频任务
- 不推荐当前阶段做此改造

---

## 我的建议

**如果预算允许（50-100 元/月）**：选 **渠道一（自助脚本部署）** 或 **渠道二（Docker Compose）**。
- 想最简单 → 渠道一
- 想以后换服务器方便 → 渠道二

**如果只想临时给别人看看效果**：选 **渠道三（Cloudflare Tunnel）**，免费且 5 分钟出链接，但依赖你电脑开机。
