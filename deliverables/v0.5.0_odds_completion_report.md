## 3. 核心算法

### 3.1 decimal 赔率 → 隐含概率

```python
decimal_to_implied_prob(2.10) == 0.4762  # 47.62%
decimal_to_implied_prob(1.50) == 0.6667  # 66.67%
```

### 3.2 归一化消除博彩公司利润(vig)

```python
remove_vig([0.5, 0.3, 0.24])  # sum=1.04 → [0.481, 0.288, 0.231]
# 总 vig = sum - 1.0 = 0.04 = 4%
```

### 3.3 价值投注率

```python
value_bet(model_prob=0.55, market_prob=0.48) == 0.146  # +14.6%
# > +5%:  强价值
# 0 ~ +5%: 边缘价值
# < 0:    模型认为被高估
```

## 6. 数据示例（实际录入）

### 6.1 match_id=14 (ESP vs CPV, 未开赛)

| Bookmaker | 主胜 | 平 | 客胜 |
|---|---|---|---|
| bet365 | 1.30 | 5.50 | 12.00 |
| pinnacle | 1.28 | 5.80 | 13.50 |
| **consensus** | **1.29** | **5.65** | **12.75** |

**consensus 市场概率（去 vig）**：
- 主胜 75.2%
- 平 17.2%
- 客胜 7.6%
- vig = 3.06%

**Elo 模型概率**：主胜 85.5% / 平 12.3% / 客胜 2.3%

**Value Bet 分析**：
- 主胜：+13.6%（**强价值**，市场低估主胜）
- 平：-28.6%（市场高估平局）
- 客胜：-70.0%（市场大幅高估客胜）

### 6.2 match_id=15 (沙特 vs 乌拉圭)

| 维度 | 市场 | Elo |
|---|---|---|
| 主胜 | 38.7% | 60.0% |
| 平 | 30.0% | 23.0% |
| 客胜 | 31.0% | 17.0% |

**Value Bet**：主胜 -61.6% / 平 +30.0% / 客胜 +189.8%（**客胜极度被低估**）
