## 📊 数据流

```
OddsSnapshot (v0.5.1)            PredictionLog (v0.6.0)
  - snapshot_at (UTC)               - predicted_at (UTC)
  - home_win/draw/away_win (dec)    - pred_home_win/draw/away_win (prob)
            ↓                                  ↓
            └──── 5 分钟内最近邻对齐 ──────────┘
                                ↓
                TimeSeriesPoint { ts, market_p, model_p }
                                ↓
                /api/odds/{match_id}/history-comparison
                                ↓
                Chart.js 双线(amber 实线市场 + cyan 虚线模型)
```

## 🆕 新增 API

```
GET /api/odds/{match_id}/history-comparison
  ?model=blend          # elo | glicko2 | blend
  &hours=72             # [1, 720], 默认 72
  &min_points=3         # 最少几个时点才返回 200

Response 200:
{
  "match_id": 1,
  "model": "blend",
  "hours": 72,
  "points": [
    { "ts": "2026-06-16T10:00:00+00:00",
      "market": {"home": 0.62, "draw": 0.22, "away": 0.16},
      "model":  {"home": 0.58, "draw": 0.25, "away": 0.17} },
    ...
  ],
  "divergence_summary": {
    "home_diff_max": 0.08,
    "draw_diff_max": 0.05,
    "away_diff_max": 0.06,
    "market_favored": "home"
  }
}

Response 204: 数据点不足 min_points
Response 404: 比赛不存在
Response 422: model 无效
```
