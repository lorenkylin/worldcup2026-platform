# v0.13.0 — Fly.io 公网一键部署

> 2026-06-17 — 距世界杯开赛 17 天

## 1. 一句话总结

> **平台正式具备公网部署能力：香港区 Fly.io + 持久卷 + secrets 管理 + 本地数据迁移。主人设置 `FLY_API_TOKEN` 后，执行 `./deploy_fly.sh` 即可获得 HTTPS 入口。**

## 2. 改动清单

### 新增文件
- `fly.toml`：Fly.io 应用配置（香港区、/data 卷、健康检查、rolling 部署）
- `deploy_fly.sh`：6 步一键部署脚本
- `fly_secrets_set.sh`：交互式注入 ADMIN_TOKEN 等 secrets
- `migrate_data_to_fly.sh`：本地 data/ 打包上传到 Fly 持久卷

### 修改文件
- `Dockerfile`：支持 `DATA_DIR` 环境变量，兼容本地 `./data` 与 Fly `/data`
- `app/main.py`：
  - `version` 改为从 git tag 动态读取（`git describe --tags --always`），防止 `/health` 版本号与 tag 脱节
  - 保留 workers=1，确保 scheduler 单实例

### 时区与数据质量（v0.13.0 后续 hotfix）
- `data/seed.py`：统一按球场真实时区转 UTC，剥离 fixture 错误偏移
- `data/seed.py`：新增 mock 赔率/快照 seed，保证 E2E 赔率页有数据
- `app/services/odds_api_client.py`：admin fetch 日期改为北京时间语义
- `tests/test_data_integrity.py`：新增 4 个回归测试，锁定 48 队 / 104 场 / 无 placeholder 结构
