## 3. 本次改动（3 处）

### 3.1 过滤 48 队（app.js exportEloToCSV）

**背景**：`/api/elo/ratings` 返 63 队（Hicruben 4 年 Elo 历史含 2026 参赛队 + 国际赛 4 年 Elo），但 `/api/teams?limit=48` 只返 48 队（2026 世界杯参赛队）。client-side join 后 15 队无 name_zh/name_en/flag_emoji，CSV 里有"半数据行"。按钮 title 写"48 队"，保持一致。

**修法**：在 `exportEloToCSV` 里加 `_has_team` 标记 + filter。

```js
const allRows = ratings.map((r, i) => {
  const t = teamMap[r.fifa_code] || {};
  return { ..._has_team: !!t.name_zh };
});
const rows = allRows.filter(r => r._has_team).map((r, idx) => ({ ...r, rank: idx + 1 }));
```
