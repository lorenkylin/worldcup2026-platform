# v0.7.8 G2-only Platt scaling — 实验性发布

> **⚠️ 实验性功能** — brier 改进未达 spec 门槛,标 experimental 警示。
> 范围: 913 场 Hicruben G2 walkforward 数据,Platt scaling 校准 (ph, pd, pa)。
> 文档: `deliverables/v0.7.8_spec.md` (147 行, A 方案: G2-only)

---

## 🎯 实验结果

| 指标 | raw G2 | calibrated G2 | Δ | spec 门槛 | 结论 |
|---|---|---|---|---|---|
| **913 场 full brier** | 0.5120 | **0.5052** | **-0.0069** (0.69 pp) | -0.015 | ⚠️ 未达 |
| **913 场 accuracy** | 0.6265 | 0.6265 | 0 | - | argmax 相同 |
| **walkforward 80/20 brier** | 0.4793 | 0.4766 | -0.0027 (0.27 pp) | -0.015 | ⚠️ 未达 |
| **walkforward 80/20 acc** | 0.6667 | 0.6557 | **-1.10pp** | - | 倒退 |
| **calibration params** | - | (a_h=-0.12, b_h=1.30, a_d=0.10, b_d=0.83, a_a=-0.04, b_a=1.20) | - | - | b>1 主队+客队 logits 放大 |

**核心结论**: calibration 让 brier 降 (calibrated 概率分布更接近真实) 但 argmax 几乎不变 (acc 持平甚至倒退)。主人 push back 后主人选 A 方案"标 experimental 推进"。

### 服务层 (calibration.py)
- `platt_fit(p_actual, p_pred) -> (a, b)` — gradient descent on brier loss (2000 步, lr=0.05)
- `platt_apply(p, a, b) -> float` — sigmoid(a + b * logit(p))
- `fit_calibrators(records) -> Calibrators` — 3 个 (a, b) 对 (home/draw/away)
- `calibrate_probs(ph, pd, pa, cals) -> tuple` — 独立校准后归一化 (sum=1)
- `walkforward_validate(records, test_ratio=0.2) -> dict` — 80/20 time-ordered split
- `load_g2_records(path) -> List[dict]` — 从 prediction_log_backfill.jsonl 读 v3_glicko2_walkforward 行
- `evaluate_all(records) -> dict` — full-sample 评估

## ⚠️ 已知风险 / 局限

1. **brier 改进 0.69 pp 低于 spec 门槛 0.015 (1.5 pp)**
2. **walkforward accuracy 倒 1.10 pp** — 校准后 argmax 极少变, 但 train 拟合时把 ph 整体下移,使原本可被 ph 拉过的临界点(>0.5)变 ≤0.5
3. **913 场 Hicruben 训练集** — 不含 2018/2022 大赛数据(v0.7.6 补 4 场未注入训练),bias 主要源于友谊赛
4. **per-team 校准不适用** — 913/48 ≈ 19 场/队,样本不够
5. **isotonic / beta calibration 未尝试** — spec 限定 Platt,后续可对比
